API Reference

Query electricity meter usage from host-owned Hinghoihome meters.

LANGUAGE:

Authentication

Include the generated API key in the x-api-key header. Do not send Public API keys with Authorization: Bearer.

x-api-key: sk_live_hh_...

Old local keys that begin with hh_live_ are no longer valid. Create a new key before testing.
